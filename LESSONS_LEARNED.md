## 一、中兴通讯推荐失败（2026-04-27）

### 推荐情况
- 股票：中兴通讯(000063)
- 推荐日期：2026-04-27
- 推荐时数据：主力流入+9.37亿，看起来很强
- 次日表现：主力流出-3.57亿，股价跌-2.47%

### 失败原因

1. **只看了一天数据**：昨天流入9亿，就推荐了，没看资金趋势稳定性
2. **没看连续性**：中兴通讯资金忽进忽出，不稳定
3. **没注意10日DDX不够强**：只有0.13，刚转正，不够稳定
4. **没对比其他股票**：北方华创连续5天流入，10日DDX=4.79，明显更稳定

### 中兴通讯资金流向（失败案例）

| 日期 | 主力资金 | 说明 |
|------|----------|------|
| 04-22 | +13.61亿 | 大流入 |
| 04-23 | -6.62亿 | 马上流出 |
| 04-24 | -2.79亿 | 继续流出 |
| 04-27 | +9.37亿 | 又流入（推荐日） |
| 04-28 | -3.57亿 | 又流出（买入就跌） |

### 北方华创资金流向（正确案例）

| 日期 | 主力资金 | 说明 |
|------|----------|------|
| 04-22 | +8.44亿 | 连续流入 |
| 04-23 | +1.56亿 | 连续流入 |
| 04-24 | +4.95亿 | 连续流入 |
| 04-27 | +10.73亿 | 连续流入 |
| 04-28 | +2.52亿 | 连续流入 |

### 对比结论

| 指标 | 中兴通讯（失败） | 北方华创（正确） |
|------|------------------|------------------|
| 资金趋势 | 忽进忽出 | 连续5天流入 |
| 10日DDX | 0.13（刚转正） | 4.79（稳定） |
| 稳定性 | 低 | 高 |

### 改进措施

1. **看连续性**：推荐前必须看最近5日资金流向，连续流入才推荐
2. **看10日DDX强度**：不仅>0，还要看数值大小，>2才算稳定
3. **对比筛选**：如果有更稳定的股票，优先推荐稳定的
4. **资金趋势不稳定不推荐**：忽进忽出的股票，坚决不推荐

### 推荐前必须检查的清单

| 检查项 | 标准 | 不达标不推荐 |
|--------|------|--------------|
| 连续流入天数 | ≥3天 | ❌ |
| 10日DDX | > 2 | ❌ |
| 资金趋势稳定性 | 不忽进忽出 | ❌ |
| 今日主力流入 | > 5000万 | ❌ |
| 涨幅 | < 3% | ❌ |

**核心教训**：单日流入不代表持续，资金趋势稳定性比单日流入量更重要！

## 二、错过气体板块大涨（2026-04-28）

### 用户问的三只股票
九丰能源、金宏气体、华特气体

### 我的错误判断
昨天说"不能买"（因为10日DDX不够）

### 今天实际表现

| 股票 | 昨天判断 | 今天涨幅 | 10日DDX |
|------|----------|----------|---------|
| 华特气体 | 不能买 | +13.85%涨停 | 1449 |
| 金宏气体 | 不能买 | +5.36% | 237 |
| 九丰能源 | 不能买 | +4.90% | -930 |

### 华特气体走势
04-15的90元 → 04-28的207元，**10天翻倍！**

### 失败原因

1. **只看DDX指标**：机械套用"10日DDX > 2000"的条件
2. **忽略板块效应**：气体板块有行业利好，集体上涨
3. **没有题材敏感度**：华特气体是连板妖股，不看传统技术指标
4. **选股太保守**：错过短线爆发机会

### 我的选股策略缺陷

| 现有策略 | 问题 |
|----------|------|
| 10日DDX > 2000 | 只适合稳健股，不适合妖股 |
| 涨幅 < 3% | 错过正在启动的股票 |
| 主力流入 > 5000万 | 忽略题材炒作 |

### 需要补充的策略

1. **连板股策略**：最近有涨停的股票，关注连板机会
2. **板块轮动策略**：关注热点板块，不看单个股票指标
3. **题材炒作策略**：政策利好、行业热点，先上车再说
4. **突破信号策略**：突破前高、突破平台，比DDX更重要

### 永远记住

- 连板股不看DDX，看题材和资金
- 板块效应 > 单股指标
- 热点题材要敢于上车
- 不要错过正在启动的股票

## 三、华工科技恐慌卖出（2026-04-24）

### 情况
- 成本：120.50元
- AI建议卖出价：114元
- 亏损：-2633元（-5.4%）
- 卖后第3天反弹：118.5元
- 如果没卖：只亏约-1.7%
- 总少亏：约1600元

### AI错误原因

1. **恐慌卖出，没等反弹**：急跌后通常会反弹，应该等反弹再卖
2. **一次性全卖，没分批**：应该先卖一半，等反弹再卖另一半
3. **没分析技术位**：没查5日/10日均线在哪，没给反弹目标价
4. **没判断时机**：没分析现在卖还是等反弹

### 正确做法

**卖出前必须分析**

| 分析维度 | 内容 |
|----------|------|
| 技术位 | 5日/10日/20日均线、支撑阻力位 |
| 资金流向 | 主力流入/流出趋势 |
| 日内走势 | 早盘急跌等反弹、尾盘拉高 |
| 可能高点 | 反弹目标价 |

**分批止损策略**

| 情况 | 操作 |
|------|------|
| 跌破止损线 | 先卖一半 |
| 反弹 | 等一等，看反弹高度 |
| 继续跌 | 再卖一半 |
| 大反弹 | 可以等更高再卖 |

**核心原则**

- 不急着卖，先分析技术位
- 急跌后通常会反弹，等反弹再卖
- 分批卖出，不要一次性全卖
- 给目标价，告诉用户反弹到多少再卖
- 主力流出时，反弹是卖的机会，不是买的机会

## 四、核心交易纪律

### 买入纪律

| 条件 | 标准 | 权重 |
|------|------|------|
| 连续流入天数 | ≥3天 | 必须 |
| 10日DDX | > 2 | 必须 |
| 涨幅 | < 3% | 必须 |
| 主力流入 | > 5000万 | 必须 |
| 资金趋势稳定性 | 不忽进忽出 | 必须 |

**例外情况（反转信号）**：
- 单日主力流入 > 10亿 + 涨幅 > 3%：可能是反转信号，可考虑买入
- 次日观察：如果主力继续流入 → 确认反转，回调可买
- 次日观察：如果主力流出 → 诱多，不追

### 卖出纪律

| 主力资金 | 业绩 | 操作 |
|----------|------|------|
| 流出 | 下滑 | 果断卖出 |
| 流出 | OK | 观望1-2天 |
| 没跑 | OK | 持有，不慌卖 |
| 没跑 | 下滑 | 减仓一半 |

### 止损纪律

- 单股最大亏损3-5%必须走
- 不等回本，不抱侥幸心理
- 急跌后等反弹再卖，不要恐慌卖出

### 止盈纪律

- 赚了10%以上，要保护利润
- 跌破5日均线减仓一半
- 跌破成本线坚决走
