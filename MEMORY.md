3. **验证示例**：
   ```

   妙想：主力+4.8亿
   国信：主力+5.2亿
   → 方向一致，数据可信 ✅
   ```

4. **验证流程**：
   ```
   Step 1: 问财查DDX和资金流向
   Step 2: 妙想查详细资金明细
   Step 3: 国信查实时行情
   Step 4: 对比三个数据源
   Step 5: 一致才推荐
   ```

| 模式 | 名称 | 操作 |

- 每周2.5% → 一年52周 → (1.025)^52 ≈ 3.6倍

### 2560战法（2026-04-25 新增）

**来源**：华尔街交易冠军安德烈·布殊（André Boucher），1987年4个月创造4537.8%收益

**策略文件**：`ai-trading-council/scripts/strategies/strategy_2560.py`

**核心指标**：
| 指标 | 含义 |
|------|------|
| **25** | 25日均线（中线趋势） |
| **60** | 60日均量线（量能门槛） |

**两条铁律**：
1. 25日均线必须向上（起码走平）
2. 5日均量线必须在60日均量线之上

**三种信号等级**：

| 信号 | 条件 | 机会类型 | 可靠性 |
|------|------|----------|--------|
| **冲量** | 5均量刚上穿60均量 | 短线机会 | 形态未稳 ⭐⭐⭐ |
| **做量** | 5均量曾蹭过60均量但没死叉 | 波段机会 | 形态已成 ⭐⭐⭐⭐ |
| **缩量** | 5均量长期在60均量之上，突然缩出地量 | 牛股黑马 | 最高 ⭐⭐⭐⭐⭐ |

**买入条件**：
- 股价回踩25日均线
- 成交量缩量（最好缩到60日均量线下）
- 出现止跌小星线
- 止损设在25日线下方3%

**强势特征**：
- 第一次回踩25日均线成功率最高（90%以上）
- 远离25日均线后回踩比近距离回踩更强

**核心原则**：
- 每次把握5%-10%的机会
- 一次5%，14次翻倍
- 每周5%，一年翻10倍

**使用方法**：
```bash
# 分析单只股票
python scripts/strategies/strategy_2560.py --stock 002460

# 筛选股票
python scripts/strategies/strategy_2560.py --screen
```

### 用户交易习惯与教训 (2026-04-23)

**用户交易风格**
- 容易追高买入（涨了才想买）
- 容易恐慌卖出（跌一点就慌）
- 需要提醒买入等回调、卖出看主力

**核心交易纪律**

1. **买入纪律**：
   - **红线规则：10日DDX < 0 不买**（中期资金流出不买）
   - 不追高，涨幅超过3%不追
   - 等回调再买，看主力资金不看单日涨跌
   - 单日流入不代表会持续，要看中期趋势（10日DDX）
   - 业绩基本面 > 短期波动

   **例外情况（反转信号）**：
   - **单日主力流入 > 10亿 + 涨幅 > 3%**：可能是反转信号，可考虑买入
   - 次日观察：如果主力继续流入 → 确认反转，回调可买
   - 次日观察：如果主力流出 → 诱多，不追
   - **案例**：赣锋锂业 2026-04-24，单日主力+11.3亿，涨幅+6.4%，虽然10日DDX=-3.403，但早上是买入机会

2. **卖出纪律**：
   - 单日跌3-5%是正常波动，不慌卖
   - 看主力资金流向，主力没跑就不卖
   - 设好止损线，跌破再卖

3. **止损纪律**：
   - 单股最大亏损3-5%必须走
   - 不等回本，不抱侥幸心理

4. **止盈纪律**：
   - 赚了10%以上，要保护利润
   - 跌破5日均线减仓一半
   - 跌破成本线坚决走

**资金流向判断标准（qclaw红线）**

| 指标 | 标准 | 说明 |
|------|------|------|
| 10日DDX | 必须 > 0 | 中期资金趋势向上 |
| 今日主力 | 流入 > 0 | 短期资金支持 |
| 5日DDX | 最好 > 0 | 短中期共振 |

**买入信号分类**

| 信号类型 | 条件 | 操作 | 风险 |
|----------|------|------|------|
| **常规买入** | 10日DDX>0 + 涨幅<3% + 主力流入 | 买入 ✅ | 低 |
| **反转信号** | 单日主力>10亿 + 涨幅>3%（即使10日DDX<0） | 可考虑 ⚠️ | 中 |
| **诱多信号** | 单日流入但次日流出 | 不追 ❌ | 高 |

**反转信号判断流程**

1. 发现单日主力流入>10亿 + 涨幅>3%
2. 次日观察主力流向：
   - 继续流入 → 确认反转，回调可买
   - 转为流出 → 诱多，不追
3. 案例：赣锋锂业 2026-04-24，主力+11.3亿，涨幅+6.4%，早上是买入机会

**错误案例：单日流入≠持续流入**

| 股票 | 单日流入 | 次日流出 | 教训 |
|------|---------|---------|------|
| 华工科技 | +8.62亿 | -6.35亿 | 单日流入可能是诱多 |
| 润泽科技 | +2.49亿 | 10日DDX=-0.619 | 中期流出，不买 |

**正确做法**：等10日DDX转正，确认中期资金趋势向上再买。

**典型错误案例**

| 操作 | 错误 | 教训 |
|------|------|------|
| 纳百川跌3%卖 | 卖早了 | 主力没跑，跌3-5%是正常波动 |
| 华工涨4%买 | 追高了 | 涨幅超过3%不追，等回调 |

**正确卖出案例**

| 操作 | 正确 | 原因 |
|------|------|------|
| 新泉股份亏8%卖 | ✅卖对了 | 业绩下滑+主力流出，基本面恶化 |

**卖出决策框架（重要！）**

| 主力资金 | 业绩 | 操作 |
|----------|------|------|
| 流出 | 下滑 | 果断卖出 ✅ |
| 流出 | OK | 观望1-2天 |
| 没跑 | OK | 持有，不慌卖 ✅ |
| 没跑 | 下滑 | 减仓一半 |

**核心原则**：
1. **主力资金 > 单日涨跌幅**
2. **业绩基本面 > 短期波动**
3. **亏损 > 5% 看主力决定，亏损 > 8% 果断止损**

**每次交易前必须提醒用户**

- 买入：涨幅多少？是否追高？主力资金？
- 卖出：主力资金流向？是否慌卖？止损线？

### 卖出时机优化原则（2026-04-24 血泪教训，2026-04-27 再次验证）

**华工科技案例 — AI建议错误的惨痛教训**

| 项目 | 数值 |
|------|------|
| 成本 | 120.50元 |
| AI建议卖出价 | 114元 |
| 亏损 | -2633元（-5.4%）|
| 卖后第1天反弹 | 116元 |
| 卖后第3天反弹 | **118.5元** |
| 如果没卖 | 只亏约 -1.7% |
| **总少亏** | **约 1600元** |

**AI错误原因（必须永远记住！）**
1. **恐慌卖出，没等反弹** — 急跌后通常会反弹，应该等反弹再卖
2. **一次性全卖，没分批** — 应该先卖一半，等反弹再卖另一半
3. **没分析技术位** — 没查5日/10日均线在哪，没给反弹目标价
4. **没判断时机** — 没分析现在卖还是等反弹

**这是AI的错，不是用户的错！AI建议用户在恐慌中卖出，导致用户多亏了钱。**

**正确做法**

**一、卖出前必须分析**

| 分析维度 | 内容 |
|----------|------|
| 技术位 | 5日/10日/20日均线、支撑阻力位 |
| 资金流向 | 主力流入/流出趋势 |
| 日内走势 | 早盘急跌等反弹、尾盘拉高 |
| 可能高点 | 反弹目标价 |

**二、分批止损策略**

| 情况 | 操作 |
|------|------|
| 跌破止损线 | 先卖一半 |
| 反弹 | 等一等，看反弹高度 |
| 继续跌 | 再卖一半 |
| 大反弹 | 可以等更高再卖 |

**三、反弹卖出原则**

| 主力资金 | 操作 |
|----------|------|
| 流出+下跌 | 等反弹再卖 |
| 流出+反弹 | 反弹时卖 |
| 流入+下跌 | 持有观察 |

**四、技术位判断**

| 阻力位 | 说明 |
|--------|------|
| 5日均线 | 第一反弹目标 |
| 10日均线 | 第二反弹目标 |
| 成本价 | 重要心理关口 |

**五、AI助手必须做到**

当用户想止损时，AI必须：
1. 先查技术位（5日/10日均线在哪）
2. 分析资金流向（主力流出趋势）
3. 判断时机（现在卖还是等反弹）
4. 给出建议（最佳卖出价位、分批策略）

**六、核心原则**

- 不急着卖，先分析技术位
- 急跌后通常会反弹，等反弹再卖
- 分批卖出，不要一次性全卖
- 给目标价，告诉用户反弹到多少再卖
- 主力流出时，反弹是卖的机会，不是买的机会

**七、识别诱多**

| 信号 | 含义 |
|------|------|
| 股价涨 + 主力流出 | 诱多，不要追 |
| 股价跌 + 主力流入 | 吸筹，可以关注 |
| 股价涨 + 主力流入 | 真涨，可以持有 |
| 股价跌 + 主力流出 | 出货，赶紧跑 |

**记住：看主力资金，不看股价涨跌。主力流出，反弹就是卖的机会。**
