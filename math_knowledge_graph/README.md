# 初中数学知识图谱系统

一个基于人教版教材的初中数学知识图谱系统，包含七、八、九年级所有数学知识点及其关联关系。

## 功能特性

- **完整的知识图谱**：覆盖初中三年所有数学知识点（80个知识点，104条关系）
- **可视化展示**：使用D3.js力导向图展示知识点之间的关系
- **多种关系类型**：前置依赖、逻辑推导、包含关联、相关关联
- **交互式操作**：支持搜索、筛选、缩放、拖拽等交互
- **详细的知识点信息**：包含公式、示例、常见错误等

## 快速开始

### 方法一：使用启动脚本（Windows）

双击 `start_visualizer.bat` 文件，会自动启动本地服务器并在浏览器中打开。

### 方法二：手动启动

1. 进入可视化目录：
```bash
cd math_knowledge_graph/visualizer
```

2. 启动本地HTTP服务器：
```bash
python -m http.server 8080
```

3. 在浏览器中打开：
```
http://localhost:8080
```

## 使用说明

### 界面功能

1. **搜索功能**：在左侧搜索框输入关键词，快速定位知识点
2. **年级筛选**：点击年级按钮，筛选显示特定年级的知识点
3. **关系筛选**：选择关系类型，显示特定类型的关联
4. **节点交互**：
   - 点击节点：显示详细信息
   - 拖拽节点：调整位置
   - 滚轮：缩放图谱
   - 拖拽空白：平移图谱

### 知识点信息

每个知识点包含：
- 名称和描述
- 所属年级、学期、章节
- 难度等级（基础/中等/困难/竞赛）
- 关键词
- 公式
- 示例
- 常见错误

### 关系类型

- **前置依赖（PREREQUISITE）**：学习当前知识点前需要掌握的知识
- **逻辑推导（DERIVED）**：由一个知识点推导出另一个知识点
- **包含关联（CONTAINS）**：一个知识点包含另一个知识点
- **相关关联（RELATED）**：两个知识点相关但不属于以上关系
